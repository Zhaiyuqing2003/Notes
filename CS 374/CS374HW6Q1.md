

```pseudocode
IsStringInL[a, b] (where a > b) # O(n^3)

mincost i (1 <= i <= n)

find all b, IsStringInL[i, b]:
	if no such b --> input is invalid (return nan)
	otherwise --> findmin (cost b + mincost b)
	# cost b --> O(1)
	# mincost b --> already calculated O(1)
	# find all b --> O(n)
	# total time O(n)
# dp goes from mincost n - 1 to mincost 1, inclusive n * O(n) = O(n^2)

total O(n^2)
```
