2 Ehrenfest Theorem

Use the Schrodinger equation to show, for a normalizable wave function, that:

d⟨p⟩dt=−⟨∂V∂x⟩

This is known as Ehrenfest's theorem, which is the quantum analog of Newton's second law.

We start from the definition of ⟨p⟩. We know that (limit omitted):

d⟨p⟩dt=−ddtiℏ∫Ψ∗∂Ψ∂xdx=−iℏ∫∂∂t(Ψ∗∂Ψ∂x)dx=−iℏ∫∂Ψ∗∂t∂Ψ∂x+Ψ∗∂Ψ∂x∂tdx

From Schrodinger equation, we know

iℏ∂Ψ∂t=−ℏ22m∂2Ψ∂x2+VΨiℏ∂Ψ∗∂t=+ℏ22m∂2Ψ∗∂x2−VΨ∗

Therefore

d⟨p⟩dt=∫∂Ψ∂x(−ℏ22m∂2Ψ∗∂x2+VΨ∗)+Ψ∗∂∂x(ℏ22m∂2Ψ∂x2−VΨ)dx=∫∂Ψ∂xVΨ∗−ℏ22m(∂Ψ∂x∂2Ψ∗∂x2−Ψ∗∂3Ψ∂x3)−Ψ∗(∂V∂xΨ+V∂Ψ∂x)dx

As noted in the book, there is a trick by integration by parts (Chapter 1, footnote 17):

∫−∞∞fdgdxdx=fg|−∞∞−∫−∞∞dfdxgdx

Since we assume that wave function and its derivative all goes to 0 when approaching infinity, we have

∫−∞∞∂Ψ∂x∂2Ψ∗∂x2dx=∂Ψ∂x∂Ψ∗∂x|−∞∞−∫−∞∞∂2Ψ∂x2∂Ψ∗∂xdx=−∂2Ψ∂x2Ψ∗|−∞∞+∫−∞∞∂3Ψ∂x3Ψ∗dx=∫−∞∞∂3Ψ∂x3Ψ∗dx

Note, as a general rule, we have

∫−∞∞∂nΨ∂xn∂mΨ∗∂xmdx=(−1)s∫−∞∞∂(n+s)Ψ∂x(n+s)∂(m−s)Ψ∗∂x(m−s)dx

given that s∈Z and Ψ and all its derivative goes to 0 as approaching infinity.

Therefore

d⟨p⟩dt=∫∂Ψ∂xVΨ∗−ℏ22m(∂3Ψ∂x3Ψ∗−Ψ∗∂3Ψ∂x3)−Ψ∗(∂V∂xΨ+V∂Ψ∂x)dx=∫∂Ψ∂xVΨ∗−Ψ∗(∂V∂xΨ+V∂Ψ∂x)dx=∫Ψ∗V(∂Ψ∂x−∂Ψ∂x)−Ψ∗∂V∂xΨdx=∫Ψ∗(−∂V∂x)Ψdx

By definition of expectation value, we see that indeed

d⟨p⟩dt=⟨−∂V∂x⟩

3 Only Energy Difference are Physical

In classical mechanics, we are taught that only differences in energy matter. We can add a constant V0 to the potential energy without changing any physics since the force F=−∇⁡V remains the same. How does the wave function change if we modify the potential energy by a constant (i.e., take V(x)→V(x)+V0)? Does this modification of the wave function change any measurable quantities?

The book actually provides a hint (Problem 1.8), that Ψ(x,t)→Ψ(x,t)exp⁡(−iV0t/ℏ) is new solution to the Schrodinger equation given that V(x)→V(x)+V0. We just need to show this is true.

So given Ψ(x,t) is a solution to Schrodinger equation with V(x),

iℏ∂∂t(Ψexp⁡(−iV0t/ℏ))=iℏ∂Ψ∂texp⁡(−iV0t/ℏ)+iℏΨ⋅(−iV0/ℏ)exp⁡(−iV0t/ℏ)=(−ℏ22m∂2Ψ∂x2+VΨ)exp⁡(−iV0t/ℏ)+ΨV0exp⁡(−iV0t/ℏ)=−ℏ22m∂2Ψ∂x2exp⁡(−iV0t/ℏ)+Ψexp⁡(−iV0t/ℏ)(V+V0)=−ℏ22m∂2∂x2(Ψexp⁡(−iV0t/ℏ))+(V+V0)(Ψexp⁡(−iV0t/ℏ))

We see that Ψexp⁡(−iV0t/ℏ) indeed solves the new Schrodinger equation with V(x)+V0. However, this doesn’t change any measurable quantities, exp⁡(−iV0t/ℏ) is independent from x. so

⟨Q′(x,p)⟩=∫Ψ∗exp⁡(iV0t/ℏ)[Q(x,−iℏ∂/∂x)](Ψexp⁡(−iV0t/ℏ))dx=∫Ψ∗exp⁡(iV0t/ℏ)exp⁡(−iV0t/ℏ)[Q(x,−iℏ∂/∂x)]Ψdx=∫Ψ∗[Q(x,−iℏ∂/∂x)]Ψ

So, we see that

⟨Q′(x,p)⟩=⟨Q(x,p)⟩

and there is no change on any measurable quantities.
